Pętla for:

Konstrukcja: for (inicjalizacja; warunek; inkrementacja) { }

Opis: Pętla `for` wykonuje określoną liczbę iteracji na podstawie inicjalizacji, warunku oraz inkrementacji.

Pętla while z kontrolowanym wejściem

Konstrukcja: while (warunek) { }

Opis: Pętla `while` wykonuje blok kodu, dopóki warunek jest prawdziwy, z kontrolowanym wejściem, co oznacza, że warunek jest sprawdzany przed wejściem do pętli.

Pętla do while z kontrolowanym wyjściem

Konstrukcja: do { } while (warunek);

Opis: Pętla `do while` wykonuje blok kodu, a następnie sprawdza warunek. Wykona się przynajmniej raz, nawet jeśli warunek jest fałszywy, z kontrolowanym wyjściem, co oznacza, że warunek jest sprawdzany po wykonaniu bloku kodu.
